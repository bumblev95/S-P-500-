S&P 500 섹터 밸류에이션 대시보드

가족용 보안 관심종목 v20 · PIN 보호 + 모바일 중심

아버지가 휴대폰에서도 쉽게 볼 수 있도록 오늘 볼 종목, 관심종목 TOP, 실적 임박, Value Trap, 부채 위험을 따로 분리해 보여주는 가족용 대시보드입니다. 가격과 펀더멘털은 GitHub Actions CSV를 읽고, 관심종목은 Google Sheet와 연결해 가족이 함께 관리할 수 있습니다.

데이터 소스 상태
기본 목록을 불러오는 중입니다. “S&P 500 목록 새로고침”을 누르면 DataHub/GitHub에서 최신 구성종목 CSV를 가져옵니다.
가격은 GitHub Actions가 생성하는 prices/latest_prices.csv의 Yahoo EOD 종가를 사용합니다. 펀더멘털/추정치는 fundamentals/latest_fundamentals.csv에서 불러옵니다. 브라우저에서 Yahoo/FMP를 직접 호출하지 않으므로 CORS와 429 문제를 피합니다. 자동으로 채워진 값이 마음에 들지 않으면 표에서 직접 수정하면 됩니다.
가격 데이터 설정
v9은 같은 GitHub 저장소의 prices/latest_prices.csv와 fundamentals/latest_fundamentals.csv를 읽습니다. 가격은 매일, 펀더멘털은 주 1회 또는 수동 실행으로 갱신하도록 구성했습니다.
4개월 모드는 모멘텀과 단기 저평가 점수를 더 크게 봅니다.
고급 가정 설정
가족용 기본값: 할인율 10%, Bear/Bull 성장률 조정 ±5%p, 적정 범위 허용치 -10%. 4개월/1년 모드는 장기 적정가보다 상대가치, 모멘텀, 리스크를 더 크게 봅니다.
